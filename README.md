## AnilistSync
Sync anilist with anidb & mal, mapping gogoanime & hianime identifiers

Example:

```
https://raw.githubusercontent.com/drxvill/anilistsync/main/data/1.json
```

Response:
```
{
    "id": 1,
    "anidb": 23,
    "myanimelist": 1,
    "title": "Cowboy Bebop",
    "mappings": {
        "gogoanime": [
            "cowboy-bebop",
            "cowboy-bebop-dub"
        ],
        "hianime": [
            "cowboy-bebop-27"
        ]
    }
}
```
